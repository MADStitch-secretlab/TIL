# 깃 사용법 정리

## git init

- 깃 초기화

```shell
git init
```

## git config

- 깃 로그인

```shell
git config --global user.email "your@email.com"
git config --global user.name "yourname"
```

## git branch

- 깃 branch 설정

```shell
git branch -M main
```

## git remote

- 깃 연결

```shell
git remote add origin https://mygithub.link
```

## git add

- 깃에 추가

```shell
git add '.\README.md'
```

## git commit

- 깃 저장 이름

```shell
git commit -m "README 작성"
```

## git push

- 깃허브 업로드

```shell
git push
```
